# coolblue-nodejs
Back-end project exercise developed to demonstrate proficiency in Node.Js TypeScript

# Technologies
- Node.js
- TypeScript
- Express
- MySQL
- Sequelize
- Docker
- Jest
- ESLint
- Prettier

# Scripts
- build: "npx tsc",
- serve: "node dist/index.js",
- dev: "concurrently \"npx tsc --watch\" \"nodemon -q ./dist/index.js\"",
